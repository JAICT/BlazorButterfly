# JAICT.Blazor.Recipes
This solution contains a sample of an Mvvm-C implementation with client-side Blazor.

## Projects in the solution

### JAICT.Blazor.Recipes.Application
Contains Application and state classes for the application.

### JAICT.Blazor.Recipes.Entities
Contains simple entities used in the recipe domain

### JAICT.Blazor.UI.Client
Contains the startup code for starting the client application.

### JAICT.Blazor.UI.Client.Mvvm
Contains a crude Mvvm library for Blazor

### JAICT.Blazor.UI.Client.ViewModels
Contains the ViewModels that together with the Application form the interaction model

### JAICT.Blazor.UI.Client.Views
Contains the Blazor Views
